Detailed Description

Derive your MonoBehaviour class from RegisteredComponent and all references to instances of this component will be saved in an internal array. Use RegisteredComponentController.GetAllOfType() to retrieve this array, which is much faster than using Unity's GameObject.FindObjectsOfType() function.
