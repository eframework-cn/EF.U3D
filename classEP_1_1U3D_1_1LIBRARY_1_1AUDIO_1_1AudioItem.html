Detailed Description

The AudioItem class represents a uniquely named audio entity that can be played by scripts.

AudioItem objects are defined in an AudioCategory using the Unity inspector.

Member Enumeration Documentation

◆ LoopMode

enum EP.U3D.LIBRARY.AUDIO.AudioItem.LoopMode

AudioItem loop mode.

Enumerator
DoNotLoop 

No looping.

LoopSubitem 

The chosen subitem (in dependence of the SubItemPickMode will be looped.

LoopSequence 

After the subitem chosen in dependence of the SubItemPickMode has stopped playing, a new subitem will be chosen and played.

Use this loop mode to generate a randomly playing looping sequence. Since Unity v4.1 this is gapless.

PlaySequenceAndLoopLast 

Play as many sub-items as specified by loopSequenceCount and loop the last one picked. Specify zero to play as many sub-items as specified in this audio item.

IntroLoopOutroSequence 

Play as many sub-items as specified by loopSequenceCount (as intro) and loop the second last one picked. If see AudioItem.Stop() is called the very last ist played as an outro.

◆ loopSequenceCount

int EP.U3D.LIBRARY.AUDIO.AudioItem.loopSequenceCount = 0

The number of sub-items to be played in the loop modes LoopMode.LoopSequence.

Specify 0 to loop infinitely (This is also the default value). In LoopMode.PlaySequenceAndLoopLast mode as many sub-item will be picked as there are sub-items specified for this audio item.

◆ loopSequenceOverlap
