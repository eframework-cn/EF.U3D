术语索引

名称 释义
平台（Platform） 渠道 + 运行时 = 平台 （默认渠道为Default，参考现有平台列表）
渠道（Channel） 分发商，如AppStore、Google Play、Steam、Oculus等
运行时（Runtime） 特指运行环境，如Windows、Android、iOS、Xbox等
版本（Version） 三段式版本号1.0.0，"1"-框架更新（二进制），"1.0"-大功能更新（二进制），"1.0.0"-资源更新（补丁热更）
打包/构建（Build） 将项目源码和资产打包为可运行程序
安装包（Archive） 可运行程序，不可热更
补丁包（Patch） 通常由资源和脚本构成，可热更

MANUAL | 使用手册

  • 快捷键（Shift+P）或菜单栏点击EFrame/Arch Ease唤起构建面板

Setting

  • 首选项和多平台设置

Operate

  • [Build Asset]：构建资源
  • [Build Script]：构建脚本
  • [Build Patch]：构建补丁包
  • [Build Patch(Push)]：构建补丁包并推送
  • [Push Patch]：推送补丁包
  • [Build Archive]：构建安装包
  • [Build All]：构建补丁包和安装包
  • [Build All(Push)]：构建补丁包和安装包并推送

Content

  • 构建后的安装包列表
  • [Path]：在文件浏览器中显示
  • [Push]：推送安装包
  • [Run]：运行安装包（安卓平台会先安装再运行，iOS平台暂不支持）
  • [Local]：本地仓库
  • [Remote]：远端仓库

构建前

  • 检查首选项配置
  • 检查多平台配置
  • 检查是否需要构建补丁包

构建时

  • 选择目标渠道和运行时
  • 点击相应的操作按钮
  • 构建完成后的包会在面板上显示

构建后

  • 点击Push可以将包推送至远端仓库以共享（完成推送后**下载链接会自动复制到粘贴板**）
