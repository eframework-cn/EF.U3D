EP.U3D.EDITOR.PREF

Preferences tool provides prefs-config's edit and display function.
Go to Scripting API

SUMMARY | 功能概述

本配置项针对于框架的一些首选项配置，包含**编辑器和运行时**两大部分，同时提供多目标选择功能，支持本地化配置，对多人协作项目较为友好，设计原理参考Unity内置的Preferences和ProjectSettings

MANUAL | 使用手册

多目标

  • Default：默认首选项配置（**会同步，慎改**）
  • Path：在目录中显示
  • Delete：删除
  • Clone：拷贝首选项配置（**新工程需要从Default拷贝一份配置**）

编辑器

  • Developer：开发者名称，用于内部工作流区分，如：补丁推送、打包构建等
  • ListenLog：监听日志服务，用于实时接收客户端的全量/异常日志
  • PushPatch：补丁包推送地址
  • PushArchive：安装包推送地址
  • Simulator Width/Height：模拟器宽/高
  • Pauseable：切换窗口后是否暂停运行
  • CheckMode：审核模式
  • ScriptBundle：脚本包模式，用于模拟真实的运行时
  • AssetBundle：资源包模式，用于模拟真实的运行时
  • CatVerbose：是否提示来自客户端推送的全量日志
  • CatException：是否提示来自客户端推送的异常日志

运行时

  • LiveMode：外网模式，线上运行环境
  • ReleaseMode：非Debug模式，屏蔽日志输出、Reporter状态显示/日志上传等**（LiveMode模式不可用）**
  • CheckUpdate：是否检查更新**（LiveMode模式不可用）**
  • ForceUpdate：是否强制更新**（LiveMode模式不可用）**
  • LogServer：日志推送地址**（LiveMode模式不可用）**
  • PatchServer：补丁下载地址**（LiveMode模式不可用）**
  • CgiServer：Cgi服务地址**（LiveMode模式不可用）**
  • ConnServer：Conn服务地址**（LiveMode模式不可用）**

操作

  • Save：保存源文件
  • Apply：**保存源文件，并应用至项目中**

FAQ | 常见问题

  • 构建时应当确认安装包的网络环境（内网/外网）
