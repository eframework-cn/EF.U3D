EP.U3D.EDITOR.PREFAB

Prefab accelerator provides prefab's browsing, editting, searching, etc.
Go to Scripting API

SUMMARY | 功能概述

本工具提供对预制的快捷操作，包含编辑预览、记忆搜索等功能，方便开发者快捷编辑UI等预制件

MANUAL | 使用手册

搜索框

模糊匹配（路径、文件名、忽略大小写），该输入框会记录上一次输入的值

内容框

显示所有的图集列表（位于Constants.PREFAB_WORKSPACE目录下，典型值为"Assets/RawAssets/Bundle/Prefab"）

  • Preview Scene: 在指定场景中预览/编辑（NONE-内置的PrefabStage工作流，TEMP-临时的场景）
  • Edit: 编辑预制
  • Path: 在文件浏览器中显示

操作框

  • Refresh: 刷新预制列表
