struct  KerningAdjustment
 Unity's dynamic font CharacterInfo struct is completely devoid of kerning-related information. In order to get dynamic fonts to print correctly, kerning information has to be retrieved from FreeType directly, then saved. This means that a part of the dynamic font is not, in fact, "dynamic", as there is no way of accessing this data outside of edit mode. More...
